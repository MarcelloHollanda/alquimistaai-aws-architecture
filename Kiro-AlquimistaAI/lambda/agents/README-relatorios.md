# Agente de Relatórios - Nigredo

## Visão Geral

O Agente de Relatórios é responsável por gerar dashboards e métricas de conversão do funil de prospecção B2B. Ele analisa dados de todos os outros agentes do Nigredo e produz insights estratégicos para otimização do processo comercial.

## Funcionalidades

### 1. Coleta de Métricas do Funil

O agente consulta dados de todas as etapas do funil:

- **Leads Recebidos**: Total de leads que entraram no sistema
- **Leads Enriquecidos**: Leads com dados complementados via MCP
- **Leads Contatados**: Leads que receberam primeira mensagem
- **Leads que Responderam**: Leads que interagiram com as mensagens
- **Leads Interessados**: Leads que demonstraram interesse real
- **Leads Agendados**: Leads com reuniões marcadas
- **Leads Convertidos**: Leads que fecharam negócio

### 2. Cálculo de Taxas de Conversão

Calcula automaticamente:

- **Taxa de Enriquecimento**: % de leads enriquecidos com sucesso
- **Taxa de Resposta**: % de leads que responderam após contato
- **Taxa de Interesse**: % de respostas que demonstraram interesse
- **Taxa de Agendamento**: % de interessados que agendaram reunião
- **Taxa de Conversão Final**: % de leads recebidos que converteram
