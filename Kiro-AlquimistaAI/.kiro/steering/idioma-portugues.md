# Configuração de Idioma - Português Brasileiro

## Regra Permanente

**SEMPRE responda em português brasileiro (pt-BR).**

Esta é uma configuração permanente do usuário e deve ser respeitada em todas as interações, incluindo:

- Respostas em chat
- Documentação gerada
- Comentários em código
- Mensagens de commit
- Nomes de arquivos de documentação
- Descrições e explicações técnicas
- Mensagens de erro e avisos
- Guias e tutoriais

## Exceções

Apenas os seguintes elementos devem permanecer em inglês:

- Código-fonte (variáveis, funções, classes)
- Nomes de arquivos de código (.ts, .tsx, .js, etc.)
- Comandos de terminal
- Configurações técnicas (JSON, YAML, etc.)
- URLs e endpoints de API
- Palavras-chave de linguagens de programação

## Exemplos

✅ **Correto:**
```typescript
// Função para validar e-mail do usuário
function validateEmail(email: string): boolean {
  // Implementação aqui
}
```

❌ **Incorreto:**
```typescript
// Function to validate user email
function validateEmail(email: string): boolean {
  // Implementation here
}
```

## Aplicação

Esta regra se aplica a:
- Todas as sessões de trabalho
- Todos os tipos de tarefas
- Toda documentação gerada
- Todas as explicações técnicas

**Não é necessário que o usuário relembre esta preferência em cada sessão.**
