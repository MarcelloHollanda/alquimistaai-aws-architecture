Núcleo central de orquestração de agentes

O Fibonacci governa usuários, subnúcleos, integrações, custos, logs e os 32 agentes especialistas da AlquimistaAI. Enquanto esta página é a interface comercial, o console técnico já roda em Lambda + API Gateway + Aurora.

  • Backend serverless na AWS (us-east-1) com banco Aurora Serverless v2.
  • Orquestração de agentes, subnúcleos e canais em um único núcleo.
  • Logs, métricas e trilhas para auditoria e LGPD.
  • Pronto para multi-tenant e comercialização como SaaS.
